#  MS Monitoring System

A professional healthcare application developed using **Flutter** and **Firebase** to assist doctors and patients in monitoring and managing **Multiple Sclerosis (MS)** cases through interactive neurological tests, patient management tools, medical dashboards, and AI-powered assistance.

---

# 📌 Overview

The project aims to provide a modern digital healthcare platform that helps:

* Doctors monitor patient progress
* Patients perform neurological tests remotely
* Organize medical sessions and reports
* Improve accessibility to healthcare services
* Support medical awareness using AI assistance

The system was designed with a strong focus on:

* Medical UI/UX
* Real-time interaction
* Clean architecture
* Scalability
* Responsive healthcare experience

---

# ✨ Main Features

# 👨‍⚕️ Doctor Dashboard

A fully interactive doctor control panel including:

* Total patients statistics
* Open sessions tracking
* Closed sessions tracking
* Notifications center
* Patient management system
* Real-time updates
* Interactive Bottom Sheets
* Search & filtering system

---

# 👤 Patient System

Patients can:

* Access neurological tests
* Upload videos and test results
* View their profiles
* Track previous activities
* Interact with the healthcare system easily

---

# 🧪 Neurological Tests Included

The system contains multiple neurological and cognitive tests including:

* Finger Tapping Test
* Finger-to-Nose Test
* Romberg Test
* Tandem Walking Test
* SDMT Test
* Memory Test
* Drawing Test
* Mood Questionnaires
* Fatigue Questionnaires
